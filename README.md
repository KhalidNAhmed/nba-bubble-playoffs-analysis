# NBA Bubble Playoffs Analysis

This project analyzes how the 2020 NBA bubble environment affected NBA playoff team performance compared to the 2019 pre-bubble playoffs.

It was originally completed for CS2316 (Data Manipulation) at Georgia Tech and uses:
- Python (pandas, numpy)
- Jupyter notebooks
- Data from Basketball Reference and the NBA API

## Files

- `PhaseII.ipynb` – Data scraping and cleaning for 2019 & 2020 playoff stats.
- `PhaseIII.ipynb` – Analysis & visualizations comparing 2019 vs 2020.
- `ONE_PAGER.pdf` – One-page summary of the project and main findings.
